Project Overview

This project is an interactive phishing awareness training website designed to teach users how to identify phishing attacks and protect themselves online. The website combines simulations, quizzes, and personalized certificates to make learning engaging and practical.

Users are guided through realistic phishing scenarios, tested with quizzes, and rewarded with a certificate of completion that automatically displays their name and score.

Features

User Registration: Users enter their full name at the start for personalized certificates.

Interactive Simulations: Four phishing simulations:

Email phishing

SMS phishing

Fake login website

Social media phishing messages

Knowledge Quiz: 8-question quiz to test understanding.

Progress Tracking: Users must complete all simulations and quizzes to unlock the certificate.

Certificate Generation: PDF certificate with user name, final score, and signature.

Responsive Design: Works on desktop and mobile with clear navigation.

Technologies Used

HTML5 – Structure of the website

CSS3 – Styling, layout, and responsive design

JavaScript – Interactive functionality for simulations, quizzes, and certificate generation

jsPDF Library – Generates downloadable PDF certificates

Installation & Setup

Clone the repository:

git clone https://github.com/YourUsername/Phishing-Awareness-Training.git


Open the project:

Open index.html in any modern web browser (Chrome, Firefox, Edge).

Optional: To host online, upload all files to a web server or GitHub Pages.

How to Use

Enter your name in the input box and click Start Training.

Complete all 4 phishing simulations by identifying suspicious elements.

Complete all 8 quiz questions. Feedback is provided immediately after each question.

Once all simulations and quizzes are completed, the certificate button unlocks.

Click Download Certificate to get a personalized PDF with your name and score.
